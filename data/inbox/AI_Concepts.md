# Artificial Intelligence Concepts

Artificial Intelligence (AI) enables machines to perform tasks that typically require human intelligence.

## Branches of AI

- Machine Learning
- Deep Learning
- Computer Vision
- Natural Language Processing
- Robotics

## Machine Learning

Machine Learning allows computers to learn patterns from data instead of following explicitly programmed rules.

### Common Algorithms

- Linear Regression
- Logistic Regression
- Random Forest
- XGBoost
- Support Vector Machine

## Applications

- Recommendation Systems
- Fraud Detection
- Medical Diagnosis
- Chatbots
- Autonomous Vehicles

## Key Terms

- Dataset
- Features
- Labels
- Training
- Inference
- Accuracy
