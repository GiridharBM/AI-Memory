# Python Programming

Python is a high-level, interpreted programming language known for its readability and simplicity. It is widely used in artificial intelligence, machine learning, web development, automation, and data science.

---

# Why Python?

Python is popular because it is:

- Easy to learn
- Cross-platform
- Open source
- Rich in libraries
- Great for automation
- Excellent for AI and Machine Learning

# Functions

```python
def greet(name):
    return f"Hello {name}"
```

Example:

```python
print(greet("World"))
```

Output:

```
Hello World
```
