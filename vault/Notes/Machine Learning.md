# machine-learning-overview

## Summary

Machine Learning is a branch of AI using labeled datasets and algorithms like Random Forest and XGBoost for applications such as fraud detection and medical diagnosis.

Machine Learning, a subset of Artificial Intelligence, involves algorithms that learn from data. Supervised Learning requires labeled datasets for training. Algorithms like Random Forest (ensemble method) and XGBoost (gradient boosting) are commonly used. These techniques find applications in fraud detection, medical diagnosis, and recommendation systems, highlighting their importance in data-driven decision making.
